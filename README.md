# AI Stylist 👗✨

AI Stylist is an AI-powered personal styling web application that provides personalized outfit recommendations based on a user's body shape and selected occasion.

## Features

- Body shape analysis
- Occasion-based outfit recommendations
- Personalized styling suggestions
- Color palette recommendations
- Virtual try-on experience
- User authentication
- Outfit collection and cart features
- Sustainable fashion suggestions

## Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite
- AI-based image and body shape analysis

## How It Works

1. User creates an account or logs in.
2. User uploads a full-body image.
3. The application analyzes body proportions and identifies the body shape.
4. User selects an occasion such as brunch, office, wedding, concert, or cocktail.
5. AI Stylist recommends suitable outfits and styling suggestions.
6. Users can explore outfit collections and preview styling options.

## Installation

Clone the repository:

```bash
git clone https://github.com/ishaa1224/ai-stylist.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```
